## overview

tRiad is a package to perform Constrained Delaunay Triangulations
([Cline and
Renka(1990)](https://epubs.siam.org/doi/abs/10.1137/0727074)) on a set
of 2D points

## installation

``` r
#Development version
#install.packages("devtools")
devtools::install_github("ZeroDawn0D/tRiad")
```
